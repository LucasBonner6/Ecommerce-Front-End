# Reserva - E-commerce Front-End

Uma reconstrução responsiva e moderna da interface principal do e-commerce da **Reserva**, focada em performance, semântica estrutural e técnicas avançadas de estilização.
Este projeto foi desenvolvido como um desafio técnico para demonstrar o domínio de conceitos modernos de Front-End Vanilla (HTML5, CSS3 de última geração e JavaScript puro).

## Funcionalidades

- **Navbar Dinâmica:** Menu superior categorizado e barra de busca integrada com ícones funcionais.
- **Hero Banner Responsivo:** Slider/Banner principal adaptável para diferentes resoluções com foco em técnicas de tipografia fluida.
- **Badges de Notificação:** Contador dinâmico no carrinho de compras.
- **Componentização Semântica:** Uso rigoroso de tags HTML5 para melhor acessibilidade e SEO.

## Tecnologias e Recursos Utilizados

O grande diferencial deste projeto foi a aplicação de recursos modernos do ecossistema Web:

- **HTML5 Semântico:** Uso de `<header>`, `<nav>`, `<main>`, `<section>` e `<figure>` para garantir acessibilidade.
- **CSS3 Avançado (Última Geração):**
- **CSS Grid & Flexbox:** Para a estrutura da malha de produtos e alinhamento milimétrico do cabeçalho.
- **Custom Properties (Variáveis CSS):** Centralização da paleta de cores da marca (vermelho Reserva, tons de cinza e fundos) facilitando a manutenção.
- **Tipografia Fluida (`clamp()`):** Ajuste automático de fontes dependendo do tamanho da tela, sem depender exclusivamente de Media Queries.
- **Nesting Nativo:** Organização do código CSS utilizando o aninhamento moderno nativo dos navegadores.
- **JavaScript (Vanilla):** Utilizado de forma pontual e eficiente para interações de interface (ex: manipulação do menu responsivo, comportamento do carrossel ou atualização do contador do carrinho).
